春學後檢討內容｜重點彙整

檢視版（現況問題與影響整理）

① 品質控管風險（人力不足＋大量產出）

品質 / 客訴 / 生產負荷
  • 在人手不足情況下需大量產出，導致品質下降，容易引發客訴。
  • 目前討論度較高品項：紅薯米糕（品質穩定度風險明顯）。
  • 近期其他品項（含活體代料／代客料理相關）亦因大量產出，難以逐一確認商品品質。
  • 現場接客與後端生產無法同時兼顧，形成兩頭燒狀況。

② 人潮高峰下補貨與銷售無法同時 Cover

現場服務 / 補貨 / 銷售體驗
  • 人潮過多時，補貨與銷售會互相排擠，同時兼顧困難。
  • 補貨期間無法完整接待來客與進行銷售說明。
  • 常出現單一人員同時面對多位客人的情況，影響服務感受與現場體驗。

③ 線上客服由現場分擔，產生觀感與效率問題

線上回覆 / 現場觀感 / 人力分工
  • 因缺乏辦公室人員協助，線上客人問題（曝光、回覆、詢問）需由現場分擔。
  • 現場人員處理訊息時，容易被顧客誤解為滑手機，造成觀感落差。
  • 同時影響現場接客專注度與回覆效率。

④ 客訂與訊息流程不穩，資訊容易中斷

客訂管理 / 訊息同步 / 訂單更新
  • 線上訊息與現場資訊不同步，常有未讀未回或訂單無法即時更新情況。
  • 客訂對話有時中斷／掉線，造成資訊銜接不完整。
  • 臨時取消訂單未能即時同步，錯過現場再銷售機會。

延伸影響

  • 取消資訊未回到現場，無法即時釋出名額／商品。
  • 可轉售給現場客人的機會流失，影響銷售轉換。

⑤ 活動溝通與優惠機制設計，影響預購意願與備量判斷

活動公告 / 現場同步 / 早鳥晚鳥機制
  • 線上已公告活動內容與現場認知不一致時，客人常到店才得知細節差異，容易產生落差。
  • 活動內容需要更早在訊息層完成溝通與同步，降低到店後爭議。
  • 目前出現「晚鳥比早鳥更優惠」情況，客人傾向延後購買，不一定搶先預約。
  • 導致前期預約量下降，備料、排產與人力配置難以準確預估。

本次彙整核心問題（主管快速檢視）

  • 人力不足導致品質控管與產出壓力並存。
  • 高峰時段補貨與銷售衝突，服務體驗下降。
  • 線上回覆與現場接客共用人力，效率與觀感受影響。
  • 客訂、取消、訂單狀態缺乏即時同步機制。
  • 活動公告與現場執行不同步，增加客訴風險。
  • 優惠機制設計（晚鳥優於早鳥）影響預購與量能預估。
